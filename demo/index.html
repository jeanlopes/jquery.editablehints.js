jQuery Editable Hints Plugin

Welcome

Feel free to type something here.

Type @ to trigger hints.

Select Google or Gorilla to insert link

select Tarzan, Titanic or Tornado for sentence

Select Papaya, pineapple or pumpkin for formated text

Usage

$.editablehints([config]);

Configuration for editor and hints must be defined

Config

editor : [Required] ID for the content editable DOM element

hints : [Required] An array of object with properties of text and value, a text is use to suggest hints as user typing, a value is use to insert into the editor when selected, can be string or HTML string

className : Class name to be use for the popup dialog, can be change if it conflict with existing class

trigger : A character that trigger the hints, defaulted to @

hintsBoxOffsetX/hintsBoxOffsetY : Additional adjustment to the popup position, negative value move it up/left
